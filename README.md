# MAFRATEC - Landing Page

Esta es una página web de landing para MAFRATEC, dedicada a la reparación de notebooks y asesoramiento informático.

## Tecnologías Utilizadas
- HTML5
- CSS3 con Tailwind CSS
- JavaScript
- GSAP para animaciones

## Secciones
- **Hero**: Introducción dinámica con animaciones.
- **Servicios**: Descripción de los servicios ofrecidos.
- **Nosotros**: Información sobre la empresa.
- **Galería**: Imágenes de trabajos con filtros por categoría.
- **Agendar Turno**: Formulario para agendar citas, con mención a sincronización en la nube.
- **Contacto**: Formulario de contacto.
- **Footer**: Información de contacto.

## Cómo Ejecutar
Abre el archivo `index.html` en tu navegador web preferido.

## Funcionalidades
- Diseño responsive.
- Animaciones con GSAP.
- Filtros en la galería.
- Formularios funcionales (alertas simples).

## Notas
- Las imágenes son placeholders. Reemplaza con imágenes reales.
- Para conectar a la nube, integra con un backend (ej. Firebase) en el futuro.
